What is Sonar?

Sonar is an open source system monitoring service.

It's useful in large business environments with lots of servers and/or workstations, and allows you to quickly monitor and manage all of them.

It's designed to be light on resources and scalable.

Goals

  • Gather CPU, RAM, and HDD usage
  • Ping hosts
  • Execute commands on remote host
  • Light footprint
  • Configurable alerts
  • Web UI
  • Flexible
  • Scalable
  • Secure

Why? Isn't this similar to XXX, YYY and ZZZ?

While similar software exists, most of it uses a dual licensing model, with more powerful features hidden behind a paywall. Sonar will be 100% Free, Libre, Open Source Software (FLOSS).

Thanks to its modular design, unlike other software, Sonar can be used to securely orchestrate servers in different datacenters.
